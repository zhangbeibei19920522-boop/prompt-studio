# 会话质检结果页紧凑化设计

## 背景

当前会话质检结果页已经具备报告式结构，但首屏仍有两处信息密度偏低：

- 顶部四个指标卡之间和卡内留白偏大
- 对话卡片折叠态存在重复引导文案，且单卡高度偏高

这次调整只做紧凑化修复，不改变结果页的信息结构、筛选逻辑和展开态详情模块。

## 目标

- 收紧顶部四个指标卡的视觉空隙，提升首屏信息密度
- 将对话卡片折叠态文案统一为“点击卡片查看对话内容与详细评估。”
- 降低对话卡片折叠态高度，使列表更紧凑

## 非目标

- 不调整顶部操作按钮区布局
- 不改指标含义、排序或数量
- 不重构对话详情展开态的双栏结构
- 不引入新的筛选或排序交互

## 设计

### 顶部指标卡

- 保持四列布局不变
- 缩小卡片之间的 `gap`
- 缩小卡片内容上边距和数字与标题之间的垂直间距

### 对话卡片折叠态

- 折叠态只保留一条引导文案：`点击卡片查看对话内容与详细评估。`
- 去掉默认摘要回退文案 `展开查看会话内容与详细评估结果。`
- 收紧卡片内容区的上下留白和标题区间距

### 展开态

- 展开后的对话内容、流程规范检查、知识问答问题模块不做结构调整

## 验证

- 顶部四卡在不改变排版结构的前提下更紧凑
- 折叠态只出现一条引导文案
- 对话卡片折叠态高度低于当前实现
- 现有详情展开内容仍可正常渲染
